## My Dotfiles:
This is my personal dotfiles configuration.
Dependencies: 
 - git
 - stow
 - fish 
 - neovim
 - alacritty
 - hyprland
 - starship 
 - rofi
 - waybar

To apply: 
```bash
$ git clone https://github.com/SmallMing6675/Dotfiles
$ cd dotfiles
$ stow --verbose --target=$$HOME --restow */ #besure to backup your existing configuration before applying!
```
